2) Which subdivision of SQL is used to insert values in tables?
=>The Data Manipulation Language (DML) subdivision of SQL is used to insert values into tables.
DML includes commands that allow you to modify data within database tables.
INSERT: Adds new rows of data into a table.
UPDATE: Modifies existing data within a table.
DELETE: Removes data from a table.
